# Todo CLI

A simple command-line todo app built with Node.js.

## Installation

1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   ```

2. Navigate to the project directory:
   ```bash
   cd todo-cli
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Link the CLI:
   ```bash
   npm link
   ```

## Usage

- Add a task:
   ```bash
   todo add
   ```

- List all tasks:
   ```bash
   todo list
   ```

- Mark a task as done:
   ```bash
   todo done <id>
   ```
